Legenda kutha

Ana ing désa-désa ing Jepang sing adoh, ana crita babagan wong wadon sing ayu banget sing kesasar ing sangisore rembulan. Sawetara wong, kanthi swara alon, wani nyebut dheweke Komayo. Sawetara crita babagan dheweke diubengi misteri lan ngomong babagan sosok sing anggun nganggo kimono biru tengah wengi, sing katon kanggo lelungan sing sepi, kaya-kaya ketarik dening kasepen. Sawetara wong ngaku yen dheweke nantang wong-wong mau kanggo main game kuna, sing aturan wis ilang, padha karo shogi. Dikandhakake yen wong-wong sing nampa undhangan iki ditemokake ing dina sabanjure, pasuryane beku kanthi eseman sing aneh, papan game ing sisihane, pikirane kaya ilang, dikonsumsi dening misteri sing ora bisa dingerteni.
